# 쇼다운 심판 앱 PRO

시각장애인 스포츠 쇼다운 경기를 위한 심판용 웹 앱입니다.

## 📱 주요 기능

### 프로젝트 관리
- ✅ 프로젝트(대회/토너먼트) 단위 관리
- ✅ 프로젝트명, 날짜, 장소 정보 저장
- ✅ 프로젝트별 경기 관리
- ✅ 경기 삭제 기능

### 경기 정보
- ✅ 선수/팀 이름
- ✅ 주심 (Main Referee)
- ✅ 부심 (Assistant Referee)
- ✅ 코치 정보 (각 선수/팀별)

### 경기 진행 (IBSA 공식 규정)
- ✅ 득점 규정
  - 골: +2점
  - 7가지 파울: 각 +1점
    - 부정 서브 (Irregular Serve)
    - 센터보드 (Centerboard)
    - 바디터치 (Body Touch)
    - 아웃 (Out of Bounds)
    - 고글터치 (Mask Touch)
    - 지연행위 (Delay of Game)
    - 2초룰 (2-Second Rule)
- ✅ 서브 순서 명확 표시 (1회차/2회차)
- ✅ 세트 종료 조건 자동 확인 (11점 이상 + 2점 차이)
- ✅ 타임아웃 기능
  - 개인전: 세트당 각 1회
  - 팀전: 전후반 각 1회 (총 2회)
  - 60초 스톱워치
  - 음성 안내: 15초 남음, 종료
- ✅ 득점 취소 기능
- ✅ 경기 기록 자동 저장

### 접근성 (시각장애인 완벽 지원)
- ✅ 5가지 시각 모드
  - 일반 모드
  - 명도 대비 강화
  - 고대비 모드
  - 색반전 모드
  - 다크 모드
- ✅ 음성 안내 (TTS)
  - 모든 동작 음성 피드백
  - 점수 변경 자동 안내
  - 세트 종료 조건 안내
- ✅ 스크린 리더 완벽 호환
- ✅ 큰 터치 영역 (최소 44x44px)
- ✅ 키보드 네비게이션 지원

## 🚀 사용 방법

### 방법 1: 로컬에서 실행
1. `index.html` 파일을 더블클릭
2. 브라우저에서 바로 실행!

### 방법 2: GitHub Pages로 배포
1. GitHub 저장소 생성
2. `index.html` 파일 업로드
3. Settings → Pages → Branch: main → Save
4. 생성된 주소로 접속!

예시 주소: `https://사용자명.github.io/저장소명/`

### 방법 3: 모바일 홈 화면에 추가
**Android Chrome:**
- 메뉴(⋮) → "홈 화면에 추가"

**iPhone Safari:**
- 공유 버튼 → "홈 화면에 추가"

## 📖 사용 가이드

### 1. 프로젝트 생성
1. 홈 화면에서 "새 프로젝트 생성" 클릭
2. 프로젝트명, 날짜, 장소 입력
3. 생성 완료!

### 2. 경기 생성
1. 프로젝트 선택
2. "새 경기 생성" 클릭
3. 경기 정보 입력:
   - 경기 유형 (개인전/팀전)
   - 세트 수 (3세트/5세트)
   - 선수 이름
   - 심판 정보
   - 코치 정보
4. 경기 시작!

### 3. 경기 진행
1. 득점 버튼 클릭
   - 골: +2점
   - 각종 파울: +1점
2. 서브 순서 자동 표시
3. 타임아웃 사용
   - 타임아웃 버튼 클릭
   - 60초 스톱워치 시작
   - 15초 남음 음성 알림
4. 세트 종료 조건 충족 시 알림
5. 세트/경기 종료

## 📝 IBSA 규정 준수

이 앱은 IBSA (국제시각장애인스포츠연맹) 쇼다운 규정 2022-2025를 준수합니다.

### 득점 규정
- 골: 2점
- 파울: 각 1점 (상대에게)

### 세트 종료 조건
- 11점 이상 득점
- 2점 이상 차이

### 타임아웃 규정
- 개인전: 세트당 1회 (60초)
- 팀전: 전후반 각 1회 (60초)
